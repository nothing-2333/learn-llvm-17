# 2.3 `Token::Token::l_paren` 错误
![alt text](image-1.png)
![alt text](image-2.png)
应该为
```c++
else 
{
    switch (*buffer_ptr) {
        #define CASE(ch, tok) \
        case ch: from_token(token, buffer_ptr + 1, tok); break
        CASE('+', Token::plus);
        CASE('-', Token::minus);
        CASE('*', Token::star);
        CASE('/', Token::slash);
        CASE('(', Token::l_paren);
        CASE(')', Token::r_paren);
        CASE(':', Token::colon);
        CASE(',', Token::comma);
        #undef CASE
        default: from_token(token, buffer_ptr + 1, Token::unknown);
    }
```
